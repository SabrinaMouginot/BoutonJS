# Bouton JavaScript stylisé

Petit projet d'entraînement consistant à créer un bouton interactif en JavaScript avec une mise en forme CSS.

## Aperçu

<img width="1867" height="825" alt="img btn" src="https://github.com/user-attachments/assets/fdf7f766-b050-4521-a87a-9d01c6020ce9" />

## Fonctionnalités

- Création dynamique du bouton en **JavaScript**
- Style entièrement réalisé en **CSS**
- Bouton circulaire
- Dégradé de couleurs
- Ombres internes et externes
- Effet de relief
- Centrage automatique sur la page

## Technologies utilisées

- HTML5
- CSS3
- JavaScript

## Structure du projet

## Lancer le projet

1. Cloner le dépôt :

```bash
git clone https://github.com/votre-utilisateur/nom-du-projet.git
```

2. Ouvrir `index.html` dans votre navigateur.

## Objectif

Ce projet est un exercice permettant de pratiquer :

- la manipulation du DOM en JavaScript ;
- la création dynamique d'éléments HTML ;
- la mise en forme avancée avec CSS ;
- les ombres, dégradés et effets visuels.
